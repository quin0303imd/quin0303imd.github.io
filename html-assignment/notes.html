Heading Tags
Heading tags are used for the headings of parts of an html website There are 6 levels of a heading tag; h1, h2, h3, h4, h5, and h6. With h1 being the largest heading and h6 being the smallest.
Paragraph Tags
Paragraph tags are use for writing paragraphs withing sections of your html site. The html tag for a paragraph is "p".
HTML Tags
The html tag is used for letting your IDE recognize where the html code is located. The html tag is put only below the doctype tag and surrounds the entirety of an html file. The html tage for html is "html".
Head Tags
The head tag is where you put all of your unseen code for example the title of the webpage. The html tag for the head is "head".
Horizontal Line Tags
The horizontal line tag is used to creat a horizontal line that puts a horizontal line through the entire page (excluding the margins). This tag can be use to divide a page up. The html tag for a horizontal line is "hr".
